# Principal Component Analysis (PCA)

PCA is an unsupervised dimensionality reduction technique that projects high-dimensional data into a lower-dimensional space while preserving maximum variance.


## Algorithm

1. Standardize the dataset (zero mean, unit variance)

2. Compute covariance matrix

3. Perform eigenvalue decomposition

4. Sort eigenvectors by descending eigenvalues

5. Select top-k principal components

6. Project data onto reduced subspace
