# Unsupervised Learning

This folder contains jupyter notebook implementations of various unsupervised learning algorithms. These models are trained on unlabeled datasets, with the aim to find patterns in the data.

Unsupervised learning is widely used for clustering and dimensional reduction problems.

---

# Algorithms Included

- PCA (Principal Component Analysis)
- DBSCAN (Density-based)
- K_Means_Clustering 
- Community_Detection (Label Propagation)

---

# Typical Workflow

- Preprocess dataset
- Train models using data
- Visualize and review patterns
