# Supervised Learning

This folder contains jupyter notebook implementations of various supervised learning algorithms. These models are trained on labeled datasets, where each input is paired with a known output. The objective is to learn a function that maps inputs to outputs and generalizes well to unseen data.

Supervised learning is widely used for both classification and regression problems.

---

# Algorithms Included

- Decision Trees
- Linear Regression
- Logistic Regression
- K-Nearest Neighbors (KNN)
- Multi-Layer Perceptron
- Perceptron
- Ensemble Methods (Bagging, Random Forests)

---

# Typical Workflow

- Preprocess dataset
- Split dataset into training and testing sets
- Train models using labeled data
- Evaluate predictions using appropriate metrics (e.g., accuracy, MSE, R2-score)
- Use trained models to make predictions on unseen data
